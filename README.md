
Markup Tool is a lightweight, fast, and intuitive annotation app designed for mobile devices. It allows users to quickly mark up images, screenshots, and documents with an easy-to-use set of drawing and editing tools.
This project focuses on precision, smooth performance, and an ergonomic UI that feels as responsive as native platform tools.

- Designers reviewing UI screenshots

- Developers marking issues

- Construction/field teams annotating site photos

- Students and professionals who need quick visual explanations
